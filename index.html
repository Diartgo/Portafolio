Proyectos Frontend

Proyecto de Pokémon con JavaScript, HTML y CSS

Este proyecto tiene como objetivo crear una aplicación interactiva de Pokémon utilizando HTML, CSS y JavaScript. La idea es permitir a los usuarios filtrar y explorar información detallada sobre los diferentes Pokémon de manera dinámica.

Desarrollo de tienda virtual en WordPress con tema Astra y WooCommerce

Se ha diseñado y desarrollado una tienda virtual utilizando WordPress con el tema Astra. Para la gestión de productos y ventas online, se ha integrado WooCommerce. Dado que el cliente solicitó una opción más precisa y adaptada para el envío, hemos desarrollado un plugin personalizado que permite sectorizar el pago según los distritos de destino.

Proyecto interactivo del Fandom de Rick and Morty: con JS, HTML, CSS y Chart.js

Este proyecto tiene como objetivo crear una página web interactiva para los fanáticos de la famosa serie Rick and Morty. Utilizando JavaScript, HTML, CSS, y la librería Chart.js, los usuarios podrán explorar los personajes y estadísticas de la serie de forma dinámica y visualmente atractiva.

Proyecto de plataforma de suscripción con validación segura de tarjetas de crédito

Superflix es una aplicación web interactiva creada con JavaScript, HTML y CSS que permite a un usuario validar el número de una tarjeta de crédito. Además, para proteger la información sensible, los dígitos de la tarjeta se ocultan, mostrando solo los últimos cuatro números.

Instapets: Red social para mascotas

Instapets es una aplicación web construida con JavaScript y Firebase que funciona como una red social para mascotas. Los dueños pueden crear perfiles para sus mascotas, postear fotos y mensajes, agregar amigos, y interactuar con otras cuentas. La plataforma permite almacenar contenido multimedia en Firebase Storage y sincronizar datos en tiempo real con Firebase Realtime Database, además de ofrecer autenticación segura con Firebase Authentication.
